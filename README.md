# 🥅 Scout Pênaltis (V2)

Aplicativo para acumular histórico de cobranças de pênalti por batedor ao longo de vários jogos.

## Funcionalidades
- Registro rápido cobrança a cobrança (com botão repetir batedor)
- Agrupamento automático por nome + número da camisa
- Perfil de cada batedor: taxa de conversão, gols/defendidos/fora, pé dominante, zonas preferidas
- Mapa de calor por jogador
- Salva automaticamente no navegador (localStorage)
- Exportar / importar histórico em JSON

## Rodar localmente
npm install && npm start

## Deploy Vercel
Push para GitHub e importe na Vercel — detecta Create React App automaticamente.
